# Linux Info Automation Tool

## 1. Introduction
A  dashboard designed for System information. This tool automates
The collection of critical system data replaces repetitive manual commands with a modular, interactive interface.

## 2. Key Features
* **Network Reconnaissance:** Instantly retrieves Public IP, Private IP, and MAC addresses.
* **Resource Monitoring:** Real-time tracking of top CPU-consuming processes and available RAM.
* **Service Auditing:** Lists active systemd services with color-coded status indicators.
* **Storage Forensics:** Recursively scans directories to identify and sort the largest files.
* **Modular Interface:** An interactive menu allows users to run specific modules or a full system scan.

## 3. Technical Skills Demonstrated
* **System Administration:** Deep integration with top, free, systemctl, and ifconfig.
* **Text Processing:** Utilizing awk and sed to filter raw command output into clean, visual reports.
* **Control Flow:** variables, custom functions, and case statements for user interaction.

## 4. Requirements
* Standard Bash shell (/bin/bash).
* Internet connection for external IP checks.
